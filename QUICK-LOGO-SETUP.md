## Step-by-Step: Using Mac Photos App

1. **Open the screenshot in Photos app**
2. Click **Edit** (top right)
3. Click the **Crop** tool
4. Drag the corners to select just the circular logo
5. Click **Done**
6. Click **File > Export > Export 1 Photo**
7. Choose **PNG** format
8. Name it `logo.png`
9. Save to the correct restaurant folder

## Using VS Code to Organize

1. Open VS Code

3. You'll see 9 folders (one for each restaurant)
4. Drag and drop each cropped logo into its folder
