## How to Crop Logos (Quick Method)

### On Mac:
1. Open screenshot in Preview
2. Click the selection tool (dotted rectangle)
3. Select the circular logo
4. Tools → Crop
5. File → Export as PNG
6. Name it `logo.png`
7. Save to the correct folder

### On iPhone (if editing on phone):
1. Open screenshot in Photos
2. Tap Edit
3. Tap Crop tool
4. Drag corners to select just the logo
5. Tap Done
6. Share → Save as File → choose folder
7. Rename to `logo.png`

## File Size & Format Tips

- **Format**: PNG preferred (transparent backgrounds work best)
- **Size**: 300x300 pixels minimum (larger is better)
- **Square ratio**: Make sure width = height
- **File name**: Must be exactly `logo.png` (lowercase)

---

## Testing After Adding Logos

1. Open `index.html` in browser
2. Scroll to "Our Partner Restaurants" section
3. All 9 logos should appear in circular frames
4. Click any restaurant card to open the menu
5. Logo should also appear in the order modal

If a logo doesn't show:
- Check spelling: `logo.png` (not Logo.png or logo.PNG)
- Check folder: Is it in the right restaurant folder?
- Hard refresh browser: Cmd+Shift+R (Mac) or Ctrl+F5 (Windows)
